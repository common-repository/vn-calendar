A simple VN calendar
== Description ==
This plugin allows you to add a simple VN calendar(lunar calendar) to widget.
You can see Can Chi days, Hoang Dao hours... in VN calendar<br\>


== Installation ==
1. Download the plugin and extract its content
2. Upload the complete folder `vn-calendar` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
